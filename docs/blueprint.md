# **App Name**: Mockup Studio AI

## Core Features:

- Logo Upload: Enable users to upload transparent PNG or SVG logos.
- Prompt Input: Accept text prompts from users describing desired mockup scenes.
- AI Mockup Generation: AI-powered tool generates 2D and 3D logo mockups based on text prompts and uploaded logo using technologies like Stable Diffusion, 3D layering, light and depth simulation, and high-fidelity rendering. The tool determines when to incorporate details to create convincing results.
- Mockup Display: Display generated mockups in a gallery layout.
- Mockup Download: Enable users to download the generated mockups.
- Responsive Design: Responsive design for desktop and mobile devices.
- Free Plan: Simple free plan offering limited functionality.

## Style Guidelines:

- Primary color: Saturated blue (#4285F4) for a studio-grade, reliable feel.
- Background color: Dark gray (#212121) for a modern, sophisticated, dark theme, with desaturation at 25%.
- Accent color: Violet (#8F00FF) to complement the blue and provide a striking contrast, analogous to blue but distinctly different.
- Font: 'Poppins', a geometric sans-serif with a precise, contemporary, fashionable, avant-garde look; suitable for short amounts of text.
- Use modern, minimalist icons relevant to mockup generation and categories.
- Clean and spacious layout with clear sections for easy navigation.
- Smooth transitions and loading animations to enhance user experience.
